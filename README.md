## Deskripsi
Aplikasi ini merupakan program yang mirip dengan aplikasi pemutar musik terkenal seperti Spotify dan lainnya dengan library lagu dan playlist lagu yang user bisa buat, menggunakan struktur data yang efisien. Program ini dibuat sebagai Tugas Besar mata kuliah Struktur Data. 

## Fitur-Fitur Aplikasi
### Admin
1. Menambah lagu ke library
2. Melihat semua lagu di library
3. Update lagu di library
4. Hapus lagu di library
5. Melihat statistik di library
### User
1. Melihat semua lagu di library
2. Mencari lagi di library
3. Play lagu dari library atau playlist
4. Next lagu dari library atau playlist
5. Prev lagu dari library atau playlist
6. Membuat playlist
7. Menambah lagu ke playlist dari library
8. Menghapus lagu di playlist
9. Melihat playlist
10. Melihat durasi total playlist
11. Shuffle Playlist
12. Menandai lagu favorit
13. Menghapus tanda favorit
14. Melihat daftar lagu favorit
